# ハンドオフ: あかブロック脱出 — ミニマルUIリデザイン（案A）

## 概要
スライドパズル「あかブロック脱出」（赤＝コーラルのブロックを下中央の出口からスライドさせて脱出させる、Unblock Me系パズル）のUIリデザイン。
既存の木目調・原色UIを、**Anthropic（Claude）サイト風の生成り＋コーラル配色**＋**角丸の日本語フォント**で、シンプル・ミニマル・「スイカゲームくらい可愛い」トーンに刷新したもの。

## このデザインファイルについて
本バンドルの HTML / JSX は **HTMLで作成したデザインリファレンス**です。見た目と意図する挙動を示すプロトタイプであり、そのまま本番コードとしてコピーするものではありません。
あなたのタスクは、**このデザインを対象アプリの既存環境（React / Vue / SwiftUI / ネイティブ等）で、その流儀・既存ライブラリに沿って再現すること**です。環境がまだ無い場合は、プロジェクトに最適なフレームワークを選んで実装してください。
（参考実装は React + インラインスタイルですが、本番では適切なCSS手法・コンポーネント分割に置き換えてOKです。）

## 忠実度
**ハイファイ（hifi）**。配色・タイポグラフィ・余白・角丸・影まで最終値を指定しています。下記の design tokens と数値を使って、できるだけ忠実に再現してください。挙動（スライド操作・タイマー等）は元アプリの仕様を踏襲します。

### コンポーネント詳細

**タイトル**
- 見出し「あかブロック脱出」: `Zen Maru Gothic` 700 / `30px` / `letter-spacing: 1px` / 色 `#2F2A22`。
  先頭にコーラルの角丸スクエア（`18×18`, `border-radius: 6px`, `background: #D9774F`, `box-shadow: 0 2px 0 #BC5A36`）を `gap: 8px` で添える。
- サブ「赤いブロックを下の出口からスライドさせよう」: `M PLUS Rounded 1c` / `12.5px` / 色 `#8B8270` / `line-height: 1.5`。

**スタッツ（3枚 / `display:flex; gap:9px`）**
- 各カード: `flex:1` / `padding: 10px 6px 11px` / `border-radius: 16px` / `background: #FFFFFF` / `border: 1px solid #EDE7DA` / 中央寄せ。
- ラベル: `11px` / 色 `#A89B82` / `letter-spacing: 1px`（「手数」「タイム」「ベスト」）。

**ボード（盤面）**
- 5列 × 6行のグリッド（`grid-template-columns: repeat(5,1fr)` / `rows: repeat(6,1fr)` / セル余白なし、各ブロックに `margin: 2px`）。`aspect-ratio: 5 / 6`。
- ボード枠: `background: #FBF9F3` / `border: 1px solid #ECE6D9` / `border-radius: 26px` / `padding: 12px` / `box-shadow: 0 10px 30px rgba(60,46,32,0.08)`。
- ブロック（角丸タイル）: `border-radius: 18px`。各ブロック上部に淡いハイライト（`linear-gradient(rgba(255,255,255,0.30), rgba(255,255,255,0))`, 高さ約32%）。
  立体感の影: `box-shadow: 0 3px 0 <edge色>, 0 6px 12px rgba(60,46,32,0.12)`（下にエッジ＋やわらかいドロップ）。
- **主役ブロックはコーラル `#D9774F`。アイコンや矢印は付けない（色だけで主役を示す）。**
- 配置データ（`{c=列, r=行, w=横スパン, h=縦スパン, t=色種}`、列・行は1始まり）:
  ```
  {c:1, r:1, w:1, h:3, t:'blue'}
  {c:2, r:1, w:1, h:2, t:'blue'}
  {c:3, r:1, w:1, h:2, t:'coral'}   ← 主役
  {c:4, r:1, w:1, h:2, t:'blue'}
  {c:5, r:1, w:1, h:3, t:'blue'}
  {c:2, r:3, w:1, h:1, t:'sand'}

  {c:4, r:4, w:1, h:1, t:'sand'}
  ```

## インタラクション & 挙動（元アプリ仕様の踏襲）
- ブロックは指のドラッグで、空きマスがある方向（縦長は上下、横長は左右）へスライド。1マス単位でスナップ。
- 主役（コーラル）が出口（下中央）から盤外へ出たらクリア。
- 1手ごとに「手数」+1、開始時から「タイム」をカウント。クリア時の手数がベスト未満なら「ベスト」更新（localStorage 等で永続化）。
- 「最初から」= 現在ステージを初期配置にリセット。「遊び方」= 操作説明モーダル。
- 推奨アニメーション: スライドは `transform: translate(...)` を `120〜160ms ease-out`。ボタン押下は `0 3px 0` のエッジ影を潰す（押し込み）表現。
- リデュースモーション設定時はアニメーション省略。

## 状態管理（必要な state）
- `blocks`: 各ブロックの現在位置（id, col, row, w, h, color, isHero）。
- `moves`（手数, number）, `elapsedSec`（経過秒, number）, `best`（ベスト手数, number|null）。
- `status`: `playing` | `cleared`。
- `helpOpen`: 遊び方モーダルの開閉。
- タイマーは `playing` 中のみ進行。
