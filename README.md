# あかブロック脱出 — スライドパズル (PWA)

色のブロックを動かす「箱入り娘（クロツキ／華容道）」型スライドパズル。
大きな **赤いブロック（2×2）** を盤の下の出口からスライドさせて脱出させればクリアです。
古典パズルの仕組みをもとにした、オリジナルの色テーマ版です。

## 遊び方
- ブロックを指（またはマウス）で空いているマス方向へスライドして動かします。
- 赤いブロックを盤の下中央の出口まで運べばクリア。最短116ステップの難問です。
- 手数・タイム・ベスト手数を記録します。

## ブロックの構成
| 色 | サイズ | 数 |
|----|--------|----|
| 赤（脱出させる） | 2×2 | 1 |
| 青 | 縦1×2 | 4 |
| 緑 | 横2×1 | 1 |
| 黄 | 1×1 | 4 |

## iPhoneでの遊び方（ホーム画面に追加）
1. このサイトを iPhone の **Safari** で開く
2. 共有ボタン → **「ホーム画面に追加」**
3. 追加されたアイコンから起動すると全画面のアプリとして遊べます（オフライン対応）

## 公開URL
https://narukuro.github.io/okinawa-trip-app/

## 構成
- `index.html` / `style.css` / `app.js` — 本体
- `manifest.json` / `sw.js` — PWA（ホーム画面追加・オフライン対応）
- `icon-*.png` — アプリアイコン
- `.github/workflows/deploy.yml` — main に push すると自動で公開される設定

## ローカルで動かす

```sh
# ブラウザで http://localhost:8000 を開く
```
