# 将棋スライドパズル (PWA)

将棋の駒をモチーフにした「箱入り娘（クロツキ／華容道）」型スライドパズル。
大きな **王将（2×2）** を盤の下の出口からスライドさせて脱出させればクリアです。
100年以上前からある古典パズル（パブリックドメイン）の仕組みをもとにした、
オリジナルの名前・デザイン版です。駒名は一般的な将棋用語です。

## 遊び方
- 駒を指（またはマウス）で空いているマス方向へスライドして動かします。
- 王将を盤の下中央の出口まで運べばクリア。最短116ステップの難問です。
- 手数・タイム・ベスト手数を記録します。

## 駒の構成
| 駒 | 色 | サイズ | 数 |
|----|----|--------|----|
| 王将 | 赤 | 2×2 | 1 |
| 飛車・角行 | 青 | 縦1×2 | 4 |
| 銀将 | 緑 | 横2×1 | 1 |
| 金 | 黄 | 1×1 | 4 |

※ 木目調（クリーム色）の見た目です。

## iPhoneでの遊び方（ホーム画面に追加）
1. このサイトを iPhone の **Safari** で開く
2. 共有ボタン → **「ホーム画面に追加」**
3. 追加されたアイコンから起動すると全画面のアプリとして遊べます（オフライン対応）

## 公開URL

## 構成
- `index.html` / `style.css` / `app.js` — 本体
- `manifest.json` / `sw.js` — PWA（ホーム画面追加・オフライン対応）
- `icon-*.png` — アプリアイコン
- `.github/workflows/deploy.yml` — main に push すると自動で公開される設定
