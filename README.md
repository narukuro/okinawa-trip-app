# あかブロック脱出 (PWA)

赤いブロックを動かす「箱入り娘（クロツキ／華容道）」型スライドパズル。
大きな **赤（コーラル）のブロック（2×2）** を盤の下の出口からスライドさせて
脱出させればクリアです。100年以上前からある古典パズル（パブリックドメイン）の
仕組みをもとにした、オリジナルの名前・デザイン版です。

UIは生成り＋コーラルのミニマル配色＋丸ゴシック（Zen Maru Gothic /
M PLUS Rounded 1c）。色だけで主役（コーラル）を示します。

## 遊び方
- ブロックを指（またはマウス）で空いているマス方向へスライドして動かします。
- 赤いブロックを盤の下中央の出口まで運べばクリア。最短116ステップの難問です。
- 手数・タイム・ベスト手数を記録します。

## ブロックの構成
| 色 | 役割 | サイズ | 数 |
|----|----|--------|----|
| コーラル | 主役（脱出させる） | 2×2 | 1 |
| ブルー | — | 縦1×2 | 4 |
| グリーン | — | 横2×1 | 1 |
| サンド | — | 1×1 | 4 |

## iPhoneでの遊び方（ホーム画面に追加）
1. このサイトを iPhone の **Safari** で開く
2. 共有ボタン → **「ホーム画面に追加」**
3. 追加されたアイコンから起動すると全画面のアプリとして遊べます

## デザイン
`design_handoff_akablock/`（案A・ミニマル）のデザイントークンに準拠。

## 構成
- `index.html` / `style.css` / `app.js` — 本体
- `manifest.json` / `sw.js` — PWA（ホーム画面追加・オフライン対応）
- `icon-*.png` — アプリアイコン
- `.github/workflows/deploy.yml` — main に push すると自動で公開される設定

## ローカルで動かす
